# 專案簡介：CaloScanAi

## 核心目標
打造一個具備帳號系統的 AI 輔助熱量追蹤 Web App，讓使用者能透過拍照（食物、熱量表、條碼）自動辨識並計算每日熱量。

## 關鍵開發策略：HTML 原型先行
- 為了確保最終體驗，將由 AI Agent 優先開發 6 種不同風格的 UI/UX (包含首頁與管理者儀表板) 的 HTML 靜態原型，部署於 Vercel。開發者挑選確認後，再接續後端與資料庫的開發。

## 關鍵功能
1. **AI 辨識**：拍照自動分析熱量，或掃描條碼/營養標示。
2. **額度管理**：可設定每日熱量上限，獨立帳號分開計算。
3. **溫暖互動**：首頁每日提供不同的鼓勵語錄 (資料庫預載 365 句)，支援深淺色模式，並加入贊助 (斗內) 按鈕。
4. **強大後台**：管理者專屬儀表板，支援總覽統計、使用者管理、以及完整的備份/還原機制。
5. **影像最佳化**：上傳前自動壓縮至 300KB 以下 (Max 1200px)。

## HTML 決策工具：拖曳卡片優先序排程器
用於與 AI 溝通任務優先序的專用網頁：
- 使用者可輸入多個工作項目
- 每個工作變成可拖曳的卡片，分為 P1/P2/P3 三個優先級
- AI 審核後給出建議順序
- 使用者可拖曳調整，最後按「確定提交」
- 網頁自動將結果轉換為一段 Prompt
- 用戶可將 Prompt 貼回給 AI，確保 AI 準確理解決策
