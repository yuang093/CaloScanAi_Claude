CaloScanAi 功能清單
==================

最後更新：2026-05-25

------------------------------------------------------------

一、會員系統
------------------------------------------------------------
1. 註冊/登入
   - 帳號密碼認證機制

2. 個人資料設定
   - 體重/身高/年齡/性別/活動量

3. 自定義 BMR
   - 可手動輸入基礎代謝率

4. 熱量目標自動計算
   - 根據 TDEE * 0.85 自動計算

------------------------------------------------------------

二、食物日誌（Dashboard）
------------------------------------------------------------
1. 圖片上傳分析
   - AI 視覺辨識食物與熱量
   - 自動儲存至食物資料庫

2. 條碼掃描
   - 拍照上傳→AI 辨識條碼
   - 自動查詢食物資料庫

3. 營養標示拍攝
   - 條碼查不到時拍攝營養標示
   - AI 解析後建立新食物資料

4. 食物資料庫搜尋
   - 可搜尋食物名稱或條碼
   - 直接加入個人日誌

5. ⭐ 我的最愛
   - 將常用食物加入最愛
   - 快速從最愛加入日誌

6. 📋 複製最近食物
   - 一鍵複製過往食物記錄
   - 省去重複拍照麻煩

7. 刪除食物記錄
   - 可刪除個別食物記錄

------------------------------------------------------------

三、每日進度
------------------------------------------------------------
1. 今日營養顯示
   - 熱量/蛋白質/碳水/脂肪

2. 進度條視覺化
   - 達成率百分比顯示

3. 熱量警示系統
   - 超過 90% 警示
   - 超過 100% 警告

4. 餐次摘要
   - 早餐/午餐/晚餐各自顯示
   - 顯示該餐食物名稱

5. 減脂建議
   - AI 根據攝取狀況給予建議

6. 每日勵志語
   - 隨機顯示鼓勵話語

------------------------------------------------------------

四、歷史統計
------------------------------------------------------------
1. 熱量趨勢圖
   - Chart.js 折線圖顯示

2. 營養素柱狀圖
   - 蛋白質/碳水/脂肪 分色顯示

3. 時間範圍選擇
   - 近一週/近一個月/近三個月

4. 平均統計
   - 平均熱量與蛋白質計算

------------------------------------------------------------

五、管理後台（Admin）
------------------------------------------------------------
1. 會員管理
   - 查看會員列表
   - 角色管理（會員/管理員）

2. 食物資料庫
   - 搜尋食物
   - 編輯食物資料（名稱/品牌/熱量/營養素）
   - 刪除食物

3. 勵志語管理
   - 新增每日勵志語
   - 刪除勵志語

4. 📦 備份管理
   - 下載 JSON 格式備份檔
   - 上傳備份檔還原資料庫

------------------------------------------------------------

六、條碼系統
------------------------------------------------------------
1. 條碼掃描
   - AI 視覺辨識條碼號碼

2. 資料庫查找
   - 找到的食物直接加入日誌
   - 找不到則引導拍攝營養標示

3. 營養標示解析
   - AI 解析營養標示圖片
   - 建立新食物資料

4. AI 分析食物
   - 拍照上傳分析熱量與營養

------------------------------------------------------------

七、贊助功能
------------------------------------------------------------
1. 斗內按鈕
   - 連結至街口支付
   - 主題：支持姜老爹

2. 費用說明
   - 用於伺服器與 API 支出

------------------------------------------------------------

八、技術架構
------------------------------------------------------------
前端：
- HTML + CSS + JavaScript
- Chart.js 圖表庫
- 無前端框架

後端：
- Node.js + Express

資料庫：
- SQLite（better-sqlite3）

AI：
- MiniMax API（圖片分析/條碼辨識）

部署：
- Docker 容器化
- VPS (Ubuntu 24.04 LTS)
- 網域：calo_c.yuang093.cc
